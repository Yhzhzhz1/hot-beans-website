Open Roles

Join us in building fast, efficient, and sustainable web experiences.

HTML Developer

As an HTML Developer, you will structure clean, semantic, and accessible web pages that form the backbone of sustainable web development.

Key Responsibilities

  • Write clean, semantic HTML5 code
  • Ensure accessibility (ARIA, screen readers)
  • Optimize page structure for performance
  • Collaborate with designers and developers

Requirements

  • Strong understanding of HTML5
  • Knowledge of SEO-friendly structure
  • Attention to detail

🌱 Sustainability Impact: Efficient markup reduces page weight, leading to faster load times and lower energy consumption.

CSS Developer

As a CSS Developer, you will design responsive and visually appealing layouts while keeping styles lightweight and efficient.

Key Responsibilities

  • Create responsive layouts
  • Maintain consistent design systems
  • Optimize stylesheets for performance

Requirements

  • Strong CSS knowledge (Flexbox, Grid)
  • Understanding of responsive design
  • Clean and maintainable code practices

🌱 Sustainability Impact: Optimized CSS reduces rendering time and improves energy efficiency across devices.

JavaScript Developer

As a JavaScript Developer, you will build interactive features while ensuring scripts are efficient and not resource-heavy.

Key Responsibilities

  • Develop interactive UI features
  • Optimize scripts for performance
  • Work with APIs when needed

Requirements

  • Strong JavaScript fundamentals
  • Understanding of DOM manipulation
  • Problem-solving mindset

🌱 Sustainability Impact: Efficient scripts reduce CPU usage, lowering energy consumption and improving performance.
